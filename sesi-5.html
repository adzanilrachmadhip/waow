Recap: Key Concepts

Client (Browser) ‚Üí sends HTTP request ‚Üí Server (Flask) ‚Üí query ‚Üí Database (MySQL)

Server process data dan kirim response balik ke client dalam format JSON.

  • GET - Read data (SELECT)
  • POST - Create data (INSERT)
  • PUT - Update data (UPDATE)
  • DELETE - Delete data (DELETE)

CRUD = Create, Read, Update, Delete

4 operasi dasar yang hampir semua aplikasi butuhkan untuk manage data.

Fetch API adalah cara modern JavaScript untuk melakukan HTTP request.

Async/await membuat code asynchronous lebih mudah dibaca (looks like synchronous code).

Bootstrap provides pre-built components (navbar, cards, tables, forms, modals) yang responsive by default.

Kita tidak perlu coding CSS dari nol!
