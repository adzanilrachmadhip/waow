Apa itu RESTful API?

REST = REpresentational State Transfer

Cara standar untuk aplikasi berkomunikasi melalui HTTP dengan format JSON.

HTTP Methods (CRUD Operations)

HTTP Method CRUD Fungsi Contoh
GET Read Ambil data GET /api/customers
POST Create Buat data baru POST /api/customers
PUT Update Update data PUT /api/customers/1
DELETE Delete Hapus data DELETE /api/customers/1

Test Installation

Mari kita test apakah Flask sudah terinstall dengan benar!

app = Flask(__name__)

@app.route('/')
def hello():
    return 'Hello, WAOW Season 6!'

    app.run(debug=True)
